PocketProject

Small ideas. Real builds. Learn by making experimental, niche, or unexpectedly useful projects.

RSVP / Interest Form → Slack (#pocketproject) →

What is this?

PocketProject is a build-focused program for learning by creating small hardware or software projects.

What you can build

tiny tools • weird websites • hardware gadgets • automations • experiments • game mods

Core idea

Not every project needs to change the world — just build something and learn from it.
